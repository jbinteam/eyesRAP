# eyesRAP Dataset Generator

A Blender-based synthetic data generation pipeline designed to create robust object detection datasets. This tool automates the process of compositing 3D scanned objects (via RealityScan) onto real-world backgrounds (VOC2012) with varied augmentations.

## 🚀 Overview

This project provides a Python script for Blender 3.6 that:
1.  **Imports** `.ply` models generated by **RealityScan**.
2.  **Composites** them onto random background images from the **VOC2012** dataset.
3.  **Augments** the scene with:
    * Random 3D rotation and position.
    * Random camera distances (Zoom).
    * Random lighting conditions (Sun direction).
    * Background resizing and shifting.
    * Edge blurring (Gaussian) to simulate depth-of-field and prevent "cut-out" artifacts.
4.  **Outputs** pairs of RGB images and binary segmentation masks.

## 🛠 Prerequisites

* **Blender 3.6 LTS** (or higher)
    * Ensure the `blender` command is added to your system PATH.
* **Python Dependencies**:
    * `PyYAML` (The script will attempt to install this automatically).
* **Data**:
    * **3D Models**: `.ply` files exported from RealityScan.
    * **Backgrounds**: [Pascal VOC 2012 Dataset](http://host.robots.ox.ac.uk/pascal/VOC/voc2012/) (`JPEGImages` folder).

## ⚙️ Configuration

Control the generation process using `config.yaml`.

```yaml
dataset:
  # Where to save the generated dataset
  output_dir: "/path/to/output/augmented_dataset"
  
  # Folder containing background images (e.g., VOC2012/JPEGImages)
  bg_dir: "/path/to/VOC2012/JPEGImages"
  
  # Number of images to generate PER OBJECT
  img_count: 50
  
  # Output resolution [Width, Height]
  resolution: [640, 480]

parameters:
  # Camera distance: 1.0 = tight fit, 2.5 = further away
  max_distance_factor: 2.5
  
  # Soften edges: 1-3 pixels (helps blend object into bg)
  edge_blur_radius: 2
  
  # Background scaling: Randomly scale bg between 1.0x and 2.0x
  bg_scale_range: [1.0, 2.0]

# List of PLY files to process
objects:
  - "/path/to/realityscan/models/object_01.ply"
  - "/path/to/realityscan/models/object_02.ply"
